# mock.js 搭建环境

> Interface configuration environment

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:2800
npm start

# 用浏览器打开 localhost:2800/get/api 是否返回
{
	code: 1000,
	message: 'get请求成功'
}
```
